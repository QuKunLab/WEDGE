Raw data download from https://github.com/czbiohub/tabula-muris
filterGeneAndCell_ClusteringRawData.R: Filter genes and cells, and generate a 10X data folder (Mouse_5W) and the clustering result of the original data (Raw.Robj)
other_Methods_Imputation: The code used by the imputation methods to impute the Tabula Muris dataset
WEDGE_Cluster.R: generate the clustering result of the WEDGE imputation data  (WEDGE.Robj)
ClusterOtherMethod.R: generate the clustering result of the data imputed by other methods
SupplementaryFig9.R generate the results of SupplementaryFig9
plot_GeneExpression.py: plot the gene expression in TSNE space
